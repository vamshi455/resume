# AI Engineer Resume Template

## Usage
Tailor `resume/master.md` for AI/ML Engineering roles by emphasizing:
- Azure AI Foundry, Azure OpenAI Service, RAG pipelines
- Multi-agent systems, MCP orchestration, Prompt Flow
- MLflow experiment tracking, model lifecycle management
- Responsible AI guardrails, model evaluation
- GitHub portfolio AI projects

## Sections to Emphasize
- GenAI/AI Engineering summary bullet
- Project #1 (Market Sensing — AI/ML signal models)
- Certifications (AB-100, Databricks GenAI, Amazon Bedrock)

## Sections to De-emphasize
- Legacy BI tools (SSIS, SSAS, SSRS, MDX)
- Earlier career projects (#11-#18)
