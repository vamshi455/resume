# Data Engineering Lead Resume Template

## Usage
Tailor `resume/master.md` for Lead/Principal Data Engineer roles by emphasizing:
- Azure Synapse, Databricks, Data Factory, Snowflake, dbt, Airflow
- Medallion architecture, Delta Lake, CDC pipelines
- Performance optimization (CTAS patterns, clustering)
- Data governance, PII masking, GDPR/CCPA
- Team leadership, cross-functional collaboration

## Sections to Emphasize
- Cloud Platforms & Data Processing summary bullet
- Data Governance & Security summary bullet
- Projects #2-#5 (Exxon VRR, Miles, KittyHawk, SAP EAM)
- Azure certifications (DP-200, DP-201)

## Sections to De-emphasize
- AI/GenAI specific tools (AI Foundry, Prompt Flow)
- Earlier career .Net development
